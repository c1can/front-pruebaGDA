## Instalacion

Instala las dependencias con npm

```bash
  npm install
```

Crea un un archivo .env en la raiz del proyect

### archivo .env

```javascript
VITE_API_URL="URL donde correra la API local"
```
### Levanta el proyecto

```bash
  npm run dev
```
